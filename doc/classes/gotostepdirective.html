Private goToStep

goToStep: WizardStep | StepOffset | number | string
  • Defined in components/directives/go-to-step.directive.ts:38

The destination step, to which the wizard should navigate, after the component, having this directive has been activated. This destination step can be given either as a WizardStep containing the step directly, a StepOffset between the current step and the wizardStep, in which this directive has been used, or a step index as a number or string
